# 列传 · 赫伯特·西蒙（Herbert Simon）

> 横跨政治学、经济学、心理学和计算机科学的通才，诺贝尔经济学奖和图灵奖的双料得主。

## 大纲

- 生平（1916-2001）
- "有限理性"（Bounded Rationality）理论
- 与 Allen Newell 合作：Logic Theorist（1955）、GPS（1957）
- Dartmouth 会议上"抢风头"——Logic Theorist 证明了数学定理
- 大胆预言："十年内计算机将成为国际象棋冠军"（错了三十年）
- 诺贝尔经济学奖（1978）、图灵奖（1975）
- 与中国的渊源：多次访华，在中国学术界有深远影响
