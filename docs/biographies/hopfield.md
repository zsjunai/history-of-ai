# 列传 · 霍普菲尔德（John Hopfield）

> 用物理学家的眼光看神经网络，发明了以自己名字命名的网络模型——六十年后，这份贡献为他赢得了诺贝尔物理学奖。

## 大纲

- 生平（1933-）
- 物理学背景：康奈尔、普林斯顿
- 霍普菲尔德网络（Hopfield Network, 1982）：用统计物理的方法理解神经网络
- 联想记忆（Associative Memory）：网络可以从部分信息恢复完整模式
- 为玻尔兹曼机（Boltzmann Machine）奠定基础
- 跨学科的典范：从物理学到神经科学到计算机科学
- 诺贝尔物理学奖（2024，与辛顿共享）
