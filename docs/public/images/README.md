# 图片资源说明

本目录存放 AI 史记中使用的图片资源。

## 目录结构

```
images/
├── people/      # 人物照片
├── events/      # 历史事件照片
└── companies/   # 公司/机构 logo 或照片
```

## 图片版权要求

所有图片必须满足以下条件之一：

1. **公共领域（Public Domain）** - 版权已过期或作者放弃版权
2. **Creative Commons 许可** - CC0、CC-BY、CC-BY-SA 均可
3. **官方媒体素材** - 公司官方提供的可自由使用的新闻图片

**不接受**：
- 未经授权的受版权保护的图片
- 从新闻网站、社交媒体直接下载的图片
- 来源不明的图片

## 推荐图片来源

- [Wikimedia Commons](https://commons.wikimedia.org/) - 最大的自由媒体文件库
- [Unsplash](https://unsplash.com/) - 免费高质量照片
- 公司官方 Press Kit / Media 页面

## 如何贡献图片

1. 确认图片许可证符合上述要求
2. 将图片放入对应子目录
3. 命名规则：使用小写英文和连字符，如 `turing.jpg`、`dartmouth-conference.jpg`
4. 建议尺寸：宽度 600-800px，格式 JPG 或 PNG
5. 在 PR 中注明每张图片的来源和许可证
