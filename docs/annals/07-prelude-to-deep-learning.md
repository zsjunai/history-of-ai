## 大纲

- 辛顿（Geoffrey Hinton）提出深度信念网络（2006）
- "深度学习"一词的回归
- GPU 计算的萌芽：从游戏显卡到科学计算
- 李飞飞（Fei-Fei Li）构建 ImageNet 数据集（2009）
- IBM 沃森（Watson）在 Jeopardy! 获胜（2011）
- Siri 发布与语音助手的兴起（2011）
- 从边缘到主流：神经网络研究者的坚守与转机
