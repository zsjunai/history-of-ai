# How to Contribute

::: warning Work in Progress
The English version is under development. See the full [Contributing Guide](https://github.com/zsjunai/history-of-ai/blob/main/CONTRIBUTING.en.md) on GitHub.
:::

We welcome all forms of contribution — corrections, new content, translations, and firsthand accounts.

## Quick Start

1. Fork the repository
2. Create a branch: `git checkout -b feat/your-topic`
3. Write or edit content
4. Preview locally: `npm run docs:dev`
5. Submit a Pull Request
