## 大纲

- **早期布局**：2006 年 AWS 推出开启了云计算时代；2015 年发布 Amazon Machine Learning，2017 年推出 SageMaker，成为企业机器学习工作流的主流平台之一
- **Alexa 与语音 AI**：2014 年 Echo + Alexa 推出，曾被视为消费级 AI 入口的代表，但随着大语言模型兴起逐步被边缘化
- **自研大模型 Titan**：2023 年起发布 Amazon Titan 系列基础模型，用于 Bedrock 平台和自有产品，但影响力弱于 GPT / Claude / Gemini
- **战略转向托管**：2023 年 4 月推出 Amazon Bedrock——一个聚合 Anthropic、Meta、Mistral、Cohere、AI21 等第三方大模型的托管 API 平台，定位"中立云上的模型超市"
- **Anthropic 投资**：2023-2024 年累计向 Anthropic 投资 **80 亿美元**，是 AWS 在 AI 时代最重要的战略押注——以资本换模型独家托管权，对标微软 + OpenAI 的深度绑定
- **Trainium / Inferentia 自研芯片**：2020 年起发布 AI 训练与推理专用芯片，与 NVIDIA 形成可控替代，Anthropic Claude 训练部分迁移到 Trainium 集群
- **AWS re:Invent AI 专场**：年度大会成为企业 AI 落地方案的发布窗口
- **Agent 与工具**：2024-2025 年推出 Bedrock Agents、Q Developer（原 CodeWhisperer），把大模型能力嵌入企业和开发者工作流
