## 大纲

- **创立**：2019 年由 Aidan Gomez（《Attention Is All You Need》作者之一，第七位署名）、Nick Frosst（辛顿的学生）、Ivan Zhang 共同创立，总部位于加拿大多伦多
- **定位差异化**：与 OpenAI / Anthropic 侧重 C 端产品和前沿模型不同，Cohere 从一开始就聚焦**企业 API + 私有化部署**，服务金融、医疗、法律等对数据安全敏感的行业
- **技术亮点**：
  - **Embed 系列**：业界领先的文本嵌入模型，RAG（检索增强生成）的核心组件
  - **Command / Command R / Command R+**：开源 / 半开源的企业级大模型，支持多语言与长上下文
  - **Rerank API**：重排序模型，提升 RAG 检索质量
- **企业合作**：与 Oracle、Salesforce、Notion、LG 等达成深度合作，是企业大模型 API 市场除 OpenAI、Anthropic 之外的最主要玩家
- **融资与估值**：2024 年完成 C 轮融资 5 亿美元，估值约 55 亿美元；投资方包括 NVIDIA、Oracle、Salesforce、加拿大政府养老金等
- **开源贡献**：旗下 Cohere For AI 研究实验室由 Sara Hooker 领导，推动开源研究与多语言模型发展（Aya 多语言模型项目）
- **加拿大 AI 的企业代表**：与多伦多大学（辛顿）、MILA（本吉奥）的学术基因一脉相承，是加拿大 AI 产业化的最重要样本
