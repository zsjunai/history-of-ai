# 世家 · 阿里达摩院（Alibaba DAMO）

> 一家电商公司斥资 1000 亿宣布建研究院的那一刻，中国科技企业第一次拥有了可与贝尔实验室对标的雄心；通义千问则把这份雄心变成了中国开源大模型最强的代表之一。

## 大纲

- **创立**：2017 年 10 月云栖大会上，马云宣布成立"达摩院"（Academy for Discovery, Adventure, Momentum and Outlook），承诺三年投入 1000 亿人民币，定位为"面向未来的基础科学研究机构"
- **首席顾问团**：奠基期邀请迈克尔·乔丹（Michael I. Jordan）、周以真、姚期智等国际顶级学者出任学术委员
- **语音识别与 NLP**：达摩院语音实验室、机器智能技术实验室，在中文语音识别、机器翻译、问答系统上长期保持国内领先
- **通义千问**：2023 年发布中文大语言模型 Qwen（通义千问），并率先将 Qwen-7B、Qwen-14B、Qwen-72B 全部开源，成为中文开源大模型的基准
- **Qwen 系列演进**：Qwen-1 → Qwen-1.5 → Qwen2 → Qwen2.5 → Qwen3，持续刷新开源模型性能上限，被海外社区（如 Hugging Face 排行榜）频繁引用
- **多模态与 Agent**：Qwen-VL（视觉）、Qwen-Audio（语音）、Qwen-Agent（工具调用），覆盖多模态与智能体生态
- **与阿里云的耦合**：依托阿里云提供模型推理与托管，打造"模型即服务"（MaaS）商业闭环，是中国云厂商中最坚定的大模型长期投入者之一
