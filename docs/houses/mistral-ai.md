# 世家 · Mistral AI

> 欧洲最具雄心的大模型公司——三位前 Meta 和 DeepMind 研究员在巴黎创立，试图用开源路线打破美国巨头对基础模型的垄断。

## 大纲

- **创立**：2023 年 4 月由 Arthur Mensch（前 DeepMind）、Guillaume Lample（前 Meta FAIR，LLaMA 核心作者）、Timothée Lacroix（前 Meta FAIR）在巴黎创立
- **种子轮即估值十亿**：2023 年 6 月首轮融资 1.05 亿欧元，创下欧洲种子轮融资纪录，估值突破 10 亿美元，被视为欧洲科技的希望
- **开源与闭源双路线**：
  - 开源：Mistral 7B（2023.9）、Mixtral 8x7B（2023.12）—— MoE 架构开源代表作
  - 闭源：Mistral Medium / Large / Le Chat 等商业化 API 与产品
- **欧洲主权 AI 代言人**：被法国总统马克龙多次公开背书，代表欧盟对"AI 主权"（AI sovereignty）的产业化回应
- **开源策略转变**：2024 年起部分后续模型转为闭源，在欧洲开源社区引发争议
- **与微软合作**：2024 年 2 月与微软达成战略合作，Mistral 模型登陆 Azure，估值进一步抬升
- **与 Meta FAIR 的血缘**：创始人团队均出自 LLaMA 项目，某种意义上 Mistral 是 LLaMA 开源精神的欧洲继承者

## 关键人物

- Guillaume Lample（CTO，LLaMA 第一作者）
