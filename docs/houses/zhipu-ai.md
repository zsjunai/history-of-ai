## 大纲

- **学院血统**：脱胎于清华大学知识工程实验室（KEG），团队核心成员多为唐杰教授的学生或合作者，长期深耕知识图谱、图神经网络与预训练语言模型
- **创立**：2019 年由清华技术团队成立，专注预训练模型研究与商业化
- **GLM 预训练范式**：2021 年提出 GLM（General Language Model Pretraining with Autoregressive Blank Infilling）预训练范式，统一理解与生成任务
- **ChatGLM 开源**：2023 年 3 月发布并开源 ChatGLM-6B，是中文社区最早可本地部署的对话大模型之一，下载量迅速破百万，推动了中文开源大模型生态
- **GLM 系列演进**：ChatGLM-6B → ChatGLM2-6B → ChatGLM3-6B → GLM-4 → GLM-4-Plus → GLM-Zero（推理模型）
- **多模态与视频**：CogVLM（视觉语言）、CogVideoX（视频生成）等开源模型在国际开源社区影响广泛
- **B 端市场**：智谱清言（ChatGLM 产品化）、MaaS 平台、私有化部署是主要商业化路径，被国内多家央企、政府机构采用
- **估值**：2024 年完成多轮融资，成为中国"大模型六小龙"中估值最高的公司之一
