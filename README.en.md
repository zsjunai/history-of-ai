# History of AI (Shiji-style)

> An open-source book on the history of Artificial Intelligence, structured in the style of *Records of the Grand Historian* (Shiji), chronicling key events, figures, and institutions from 1943 to 2026 using the classical five-part format.

## Read Online

🔗 **https://zsjunai.github.io/history-of-ai/**

## Project Scope

| Section | Count | Description |
|---------|-------|-------------|
| Annals | 9 | Chronological narratives recording the core debates and turning points of AI development by era |
| Houses | 14 | Institutional histories, from the MIT AI Lab to OpenAI to Tsinghua University |
| Biographies | 23 | Personal biographies, from Turing and Shannon to Hinton and Altman |
| Treatises | 25 | Technical topics + national histories, covering neural networks to AGI |
| Timeline | 1 | A chronological table of key events across 83 years of AI |
| **Total** | **72** | |

## Five-Part Structure

### Annals

Nine chronological narratives centered on intellectual conflicts:

| # | Title | Period |
|---|-------|--------|
| 1 | The Dawn | 1943-1956 |
| 2 | The Golden Age | 1956-1973 |
| 3 | The First AI Winter | 1973-1980 |
| 4 | The Expert Systems Era | 1980-1987 |
| 5 | The Second AI Winter | 1987-1993 |
| 6 | The Silent Advance | 1993-2012 |
| 7 | The Deep Learning Revolution | 2012-2017 |
| 8 | The Transformer Era | 2017-2022 |
| 9 | The Generative AI Wave | 2022-2026 |

### Houses

14 key institutions that shaped AI history:

MIT AI Lab · CMU · Stanford AI Lab · Bell Labs · IBM Research · MILA · DeepMind · OpenAI · Google Brain · Anthropic · Meta AI (FAIR) · xAI · Baidu AI · Tsinghua University AI

### Biographies

23 key figures, grouped by era:

**Founders**: Turing · Shannon · Wiener · McCarthy · Minsky · Simon · Newell · Rosenblatt · Pearl

**Deep Learning Pioneers**: Hopfield · Hinton · Schmidhuber · LeCun · Bengio

**Contemporary Leaders**: Fei-Fei Li · Andrew Ng · Sutskever · Hassabis · Altman · Karpathy · Musk

**Chinese Figures**: Kai-Fu Lee · Zhihua Zhou

### Treatises

25 thematic essays organized into six groups:

**Technical Threads**: Neural Networks · Knowledge Representation & Reasoning · NLP · Computer Vision · Reinforcement Learning · Large Language Models · World Models · AGI & ASI

**Infrastructure**: AI Hardware & Compute · AI & the Open Source Movement

**Application Domains**: AI & Games · AI & Biomedicine · AI & Creative Arts · Embodied Intelligence & Robotics · Autonomous Driving · AI Agents

**Social Impact**: AI Ethics · AI Safety & Alignment · AI Policy & Regulation · AI & Web3

**National Histories**: United States · China · Europe · Canada · Japan

### Timeline

A chronological table of AI milestones from 1943 to 2026, with each entry linking to the corresponding chapter.

## Features

- 📜 **Shiji-style Structure**: Five-part format (Annals / Houses / Biographies / Treatises / Timeline) for a multi-dimensional record of AI history
- 💬 **Historian's Commentary**: A subjective author's commentary at the end of each chapter — no pretense of objectivity
- 🎙️ **Firsthand Accounts**: Each chapter reserves a section for community oral histories, inviting firsthand contributors
- 🔗 **Open-Source Collaboration**: Collaborate on GitHub, report errors via Issues, contribute via PRs
- 🌐 **Bilingual (Chinese & English)**: Primarily in Chinese, with key terms preserved in their original English

## Tech Stack

- [VitePress](https://vitepress.dev/) static site generation
- Custom tech-inspired theme (cyan-blue + purple gradient)
- Particle neural network background animation
- Historical loading terminal animation (typewriter effect)
- AI robot interactive animation
- GitHub Actions auto-deployment to GitHub Pages
- Shared timeline data source (`timeline.ts`)

## Local Development

```bash
# Clone the repository
git clone https://github.com/zsjunai/history-of-ai.git
cd history-of-ai

# Install dependencies
npm install

# Start local preview (with hot reload)
npm run docs:dev

# Build
npm run docs:build

# Preview the build
npm run docs:preview
```
